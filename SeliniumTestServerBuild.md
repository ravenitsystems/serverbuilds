# Selinium Based Test Server

This build creates a server with all the tools needed to do automated browser based testing, made easier by the BHAT plugin which makes writing tests simple. 

## Base System

The server this is built on is AlmaLinux 10 however Centos 10 will be almost identical, I know most builds use AlmaLinux 8 but I did not want to complicate chrome driver installs with an older OS.
